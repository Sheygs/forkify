## Tasty

> A food recipe search web app :stuck_out_tongue:

## Installation

```shell

   # Clone the repository
   git clone https://github.com/sheygs13/forkify.git

   # Change directory
   cd forkify

   # Install dependencies
   npm install

   # Run your app
   npm start

   # Minify your app
   npm build

   # Run in development
   npm run dev

```
